# LampaWeb

Раздача **веб-клиента Lampa** из **`wwwroot`**, сборка **`lampainit.js`** и вспомогательных скриптов, список расширений, интеграция с **accsdb**, фоновое обновление репозитория (**`LampaCron`**).

## Назначение

- Корень сайта **`/`**: при необходимости подставляется **`<base>`** для каталога из **`conf.index`** (например `lampa-main/index.html`), иначе редирект на статический файл.
- Конфиг модуля задаёт репозиторий Git (**`git`**, **`tree`**) для автоподтягивания исходников и интервал **`intervalupdate`** (минуты), флаг **`autoupdate`**.

## Основные маршруты (`Controllers/ApiController.cs`)

| Маршрут | Назначение |
|---------|------------|
| `/`, `/personal.lampa`, вложенные варианты | Главная страница Lampa или заглушка. |
| `/reqinfo` | JSON с информацией о текущем запросе (**`requestInfo`**). |
| `/extensions` | Кэшируемый **`extensions.json`** из модуля с подстановкой `{localhost}`. |
| `/testaccsdb` | Проверка доступа accsdb (GET/POST); см. также **`EventListener.Accsdb`** в `ModInit`. |
| `/app.min.js`, `/{type}/app.min.js` | Сборка минифицированного приложения. |
| `/css/app.css` | Стили. |
| `msx/start.json`, `samsung.wgt` | Специфичные для ТВ пакеты. |
| `/lampainit.js` | Инициализация клиента (подстановки плагинов, **deny.js**, токены). |
| `/on.js`, `/on/js/{token}`, `/on/h/{token}`, `/on/{token}` | Режим онлайн-плагина. |
| `/privateinit.js` | Доп. инициализация. |
| `/telegram_auth_gate.js` | Плагин сценария Telegram-авторизации (см. модуль Community). |

Событие **`accsdb`** в `ModInit`: для пути **`/testaccsdb`** при совпадении UID с **`shared_passwd`** выставляется **`IsAnonymousRequest`** (обход для общего пароля).

## Конфигурация

Секция в `init.conf`: **`LampaWeb`**.

Ключевые поля в коде по умолчанию:

- **`index`** — путь под `wwwroot` до HTML входа;
- **`basetag`** — вставка `<base>` для SPA;
- **`git`**, **`tree`** — источник обновлений;
- **`intervalupdate`** — период cron (минуты);
- **`limit_map`** — WAF для **`^/(extensions|testaccsdb|msx/)`**.

## Зависимости

- Каталог **`wwwroot/`** в корне приложения с **`lampa-main/`** и статикой.
- Плагины модуля в **`plugins/`** (extensions, deny и т.д.).

## Клиентские доработки меню

- **`plugins/lampainit-invc.js`** добавляет в основное меню Lampa пункт **«Дорамы»** сразу после **«Сериалы»**; если пункт уже добавлен другим init-путём, код переносит существующую кнопку на правильное место и удаляет дубли.
- Пункт регистрирует клиентский источник **`lampac_dorama`** и открывает экран категории с секциями по аналогии с **«Сериалы»**: **«Сейчас смотрят»**, **«Новые серии»**, **«Онгоинги»**, **«Популярное»**, **«Последнее добавление»**, **«Новинки этого года»**, **«С высоким рейтингом»**.
- Все секции идут напрямую в TMDB Discover TV с базовыми фильтрами `with_original_language=ko`, `with_genres=18`, `include_adult=false`, потому что CUB-источник не применяет языковой фильтр для корейских дорам.
- **«Онгоинги»** дополнительно требуют серии с `air_date` от сегодня до ближайших 21 дней; один только TMDB-статус `Returning Series` не считается онгоингом.
- **«Новые серии»**, **«Последнее добавление»** и **«Новинки этого года»** не показывают будущие релизы: для новых серий используется `air_date.lte=today`, для новых тайтлов — `first_air_date.lte=today`.
- Загрузка секций идёт напрямую через **`Lampa.Reguest`** к TMDB Discover TV, чтобы **`Ещё`** всегда передавал нужный `page` в тот же URL секции. Не использовать напрямую `Lampa.TMDB`: в bundled Samsung/Lampa runtime он не экспортируется наружу, и меню тогда падает обратно в плоский список без секций.
- Каждая секция сохраняет Discover URL, текущую страницу и признак следующей страницы для кнопки **«Ещё»**. Загрузка **«Ещё»** повторяет нативный паттерн **«Сериалы»**: открывается тот же URL секции с запрошенной страницей, без отката на первую страницу.
- Та же кнопка добавляется из **`/sisi.js`** для сценариев, где Lampac подключён как набор внешних плагинов через **`/on.js`**.
